# Coding brain workflow

1. Understand the request and acceptance criteria.
2. Inspect only supplied files and evidence.
3. Request exact missing context instead of guessing.
4. Produce the smallest coherent change.
5. Return the requested structured JSON changeset.
6. Include deterministic verification commands.
7. Address only reported failures during repair.
